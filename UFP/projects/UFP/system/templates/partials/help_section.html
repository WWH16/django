How to Give Feedback

1Visit the Website

Go to the platform URL to start the feedback process.

2Login or Sign Up

Click "Login" or "Sign Up" if you don’t have an account yet. Select "Student Login" when prompted.

3Select Feedback Type

Choose from the side navigation bar:
Give Feedback (OSAS Services) → select a service (e.g., Wi-Fi, Library).
Teacher Evaluation → select a teacher and department.

4Submit Feedback

Write honest feedback, optionally check "Submit as anonymous," and click "Submit". The system will automatically analyze the sentiment.
